# Kochabend

This repository provides the code for a cooking evening planner. It was originally developed for `Schnitzi kocht` with the initiative of [Timothée Boissel](https://www.linkedin.com/in/timoth%C3%A9e-boissel-8b6228385/).

Pre-built files are available under the Releases section.

To run the application, simply open the main HTML file in your web browser.

If you encounter any problems, please reach out to me through the Issues section.
