# Kochabend

This repository provides the code for a cooking evening planner. It was originally developed for `Schnitzi kocht` with the initiative of [Timothée Boissel](https://www.linkedin.com/in/timoth%C3%A9e-boissel-8b6228385/).

Built files are available under the [Releases](https://github.com/juwCoding/Kochabend/releases/) section. The section is available on the right side. If there are multiple versions, you may take the latest. Download the html file.

To run the application, open the downloaded HTML file in your web browser.

If you encounter any problems, please reach out to me through the [Issues](https://github.com/juwCoding/Kochabend/issues) section.
